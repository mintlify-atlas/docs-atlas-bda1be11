      Quick Start
    </h2>
    <p className="text-lg text-gray-600 dark:text-gray-400">
      Get your TaskFlow application up and running in minutes
    </p>
  </div>

  <Steps>
    <Step title="Clone the repository">
      Clone the TaskFlow Angular repository to your local machine:
      
      ```bash
      git clone https://github.com/JoelVM2/TaskFlowAngular.git
      cd TaskFlowAngular
      ```
    </Step>
    
    <Step title="Install dependencies">
      Install all required npm packages including Angular and Angular CDK:
      
      ```bash
      npm install
      ```
      
      This will install Angular 21, Angular CDK for drag-and-drop, RxJS, and all development dependencies.
    </Step>
    
    <Step title="Configure the backend API">
      Update the API URL in your service files to point to your backend:
      
      ```typescript
      // In src/app/services/*.service.ts
      private apiUrl = 'https://localhost:7057/api';
      ```
      
      <Note>
        TaskFlow requires a .NET backend API. Ensure your backend is running before starting the frontend.
      </Note>
    </Step>
    
    <Step title="Start the development server">
      Run the Angular development server:
      
      ```bash
      ng serve
      ```
      
      Your application will be available at `http://localhost:4200`
    </Step>
  </Steps>
</div>
